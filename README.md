# Ethcoin Miner

Desktop mining client for [Ethcoin](https://ethcoin.org) on Ethereum mainnet.

## Features

- One-click mining with adjustable mining power
- Built-in wallet (create, import, or restore)
- Real-time mining stats and history
- Send/receive ETH and ETHC
- Auto-pause when gas price exceeds your limit
- System tray support — mines in the background
- Auto-update notifications
